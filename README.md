**`AFOnboarding`** is a SwiftUI-based iOS app that provides an engaging onboarding experience.

# Getting Started

### Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

### Installation

1. Clone the repository:
``` bash
git clone https://github.com/gurelyusuf/AF-Onboarding.git
```
